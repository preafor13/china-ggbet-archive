# China GGBet Archive

## 项目简介

本仓库用于归档与发布多个独立 HTML 页面。这些页面以静态文件形式存储，内容涵盖不同主题和用途，不针对任何单一域名或具体网站。仓库旨在提供一种结构化的方式来保存和共享网页快照及离线页面。

## 目录说明

```
china-ggbet-archive/
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库首页或页面索引（可选）
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件名对应一个独立的归档页面。
- **assets/**：按需存放页面所需的公共或私有资源。
- **index.html**：可选的导航或索引页面，方便浏览所有归档页面。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开使用。
- 页面内容可能包含文本、图片、样式及脚本，但不会引用外部不可控资源（除非必要）。
- 归档时尽量保留原始页面的结构与功能，但可能会对部分元素进行调整以确保本地或静态环境下的兼容性。
- 页面命名遵循 `YYYY-MM-DD-description.html` 的格式，以便按时间排序和识别。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面时会提交明确的 commit 信息。
- 欢迎通过 Issues 提交页面归档建议或报告链接失效等问题。
- 不接受直接向仓库提交包含敏感或侵权内容的页面。
- 所有归档页面仅供学习、研究或备份目的使用，请勿用于商业或非法用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 `index.html`（如存在）浏览所有页面列表。
